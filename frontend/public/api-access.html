Access the Amazon Ads API

If you’re an admin, you can apply for and access the Amazon Ads API.

To link your Login with Amazon (LWA) application to the Partner Network, the signed-in user must have admin user role on both the Partner Network and the LWA app. Associates can’t apply for or link existing API credentials.

You’re not required to apply for Amazon Ads API access to complete your Partner Network registration. If your company already has API access, linking your Partner Network account will allow automatic mapping of your advertisers and ensure we use the correct information when assigning product verifications.

To apply for Amazon Ads API access

  1. Create your Login with Amazon application
  2. Apply for Amazon Ads API access — this can be done in the Partner Network.
  3. Assign API access to a Login with Amazon application
  4. Create an authorization grant

After that, you can use the Amazon Advertising API docs and configure this app with your LWA Client ID, Client Secret, and Redirect URI.

Troubleshoot the Amazon Ads API

Solve common issues with the Amazon Ads API (support overview).

I was denied access to the Amazon Ads API. How do I appeal?

Review the Amazon email that explains the reason for denial and the steps to resolve it. In some cases, access may not be granted even after an appeal. Review the Data Policy before appealing to ensure compliance with terms and conditions. If you need support, contact Amazon through your JIRA support account.

I already have access to Amazon Ads APIs through a different account. How do I get access without re-applying?

To get multiple user access:

  • Go to Account access & settings under the Administration menu.
  • On the Users tab, select Invite user.
  • Invite the account that already has API access. Enter that user’s name and email, set access to Admin, and click Send invite. Use the same email that previously obtained API access.
  • The invited account must sign in with that email (the one that already has API access).
  • Once signed in, you can link API access via the Amazon Ads API tab by selecting Link application. Only users with admin access in both the Partner Network and the LWA app can do this.

I need Amazon Ads API support. Who can I contact?

An admin of your Partner Network account should contact support via the support portal. Submit requests using the email that was registered for API access.

Important: Email safe sender list

Add these addresses to your email safe sender list so Amazon’s messages are not marked as spam:

  • ads-api-onboarding@amazon.com
